# PlanIt

A local event discovery app. See [docs/planit_mvp.md](docs/planit_mvp.md) for full specification.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack

- **Frontend:** Next.js 14 (App Router)
- **UI:** Material UI v5
- **Maps:** Leaflet + React-Leaflet
- **Backend/DB:** Supabase (Postgres, Auth, Storage)
- **Hosting:** Vercel
